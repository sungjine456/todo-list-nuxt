# todo-list-nuxt

Nuxt에 대해 가볍게 사용해보며 만들어보는 Todo List.

## 사용한 기술 목록

- Nuxt 3 - Composition API
- SCSS
- Typescript

## Nuxt 공식 문서

- [Nuxt 3 documentation](https://nuxt.com/docs/getting-started/introduction)
- [Deployment documentation](https://nuxt.com/docs/getting-started/deployment)

## Setup

```
npm install
```

## Development

개발 포트: `3000`

```
npm run dev
```

## Production

```
npm run build
```

```
npm run preview
```
